# Garden City Painters

A professional mockup for Garden City Painters, built with Next.js and Tailwind CSS.

## Features

- Responsive design
- Modern aesthetics
- Service-oriented sections
- Contact information and trust signals

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion (for animations)

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
